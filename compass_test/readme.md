compass_test
============

Example code for the
[CMPS10](http://www.robot-electronics.co.uk/htm/cmps10doc.htm)
tilt-compensating compass. Displays the bearing in a continuous loop.

Wiring
======

* TX -> A4
* RX -> A5 
* Both have 3.3k pullup resistors.
